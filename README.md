# HeartVector — website

One-page marketing site for **HeartVector**, a real-time heart-rate-zone training app.

- Single self-contained `index.html` (inline CSS + SVG logo; fonts from Google Fonts).
- Hosted on **GitHub Pages**; custom domain set via `CNAME` (DNS managed in Porkbun).
- Theme matches the app: black background, red `#FF3B30` primary, orange `#FF6A00`
  accent, and the five HR-zone colors.

App repo: https://github.com/pechavarria/HeartVector
